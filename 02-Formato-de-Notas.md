# 📝 Formato de Notas

Tres plantillas: **apuntes de clase**, **bitácora diaria** y **revisión semanal**.
Copia y pega la que necesites. Mantenlo simple: un sistema que no se usa no sirve.

---

## 1) Apuntes de clase (método Cornell adaptado)

```
Materia: _______________________   Fecha: ____________
Tema: __________________________   Semana #: ____

┌──────────────────┬─────────────────────────────────────────┐
│  PALABRAS CLAVE  │  NOTAS                                   │
│  / PREGUNTAS     │  (ideas, fórmulas, ejemplos)             │
│                  │                                          │
│  · ...           │  - ...                                   │
│  · ...           │  - ...                                   │
│  · ¿duda?        │  - ...                                   │
└──────────────────┴─────────────────────────────────────────┘

RESUMEN (3–5 líneas, con mis propias palabras):
> ...

⚡ ACCIÓN / TAREA derivada: __________________________________
🔁 Repasar el: __________ (regla: repaso 1 día, 7 días y 30 días después)
```

> **Por qué funciona:** la columna de la izquierda te obliga a hacerte preguntas (estudio activo) y el resumen final consolida la memoria. El repaso espaciado (1-7-30) es lo que de verdad fija el conocimiento.

---

## 2) Bitácora diaria (2 minutos, en la noche)

```
Fecha: ____________   Energía (1-5): ___   Ánimo (1-5): ___

🎯 Mis 3 prioridades de hoy:  [✓/✗]
   1. ____________________________  [ ]
   2. ____________________________  [ ]
   3. ____________________________  [ ]

✅ Hábitos:  Inglés[ ]  Gym/Box[ ]  Lectura[ ]  Proyecto[ ]  Agua/Comida[ ]

😊 1 cosa buena de hoy: ______________________________________
🧠 1 cosa que mejoraría mañana: ______________________________
```

---

## 3) Revisión semanal (domingo, 15 min)

```
Semana del ___ al ___

📈 ¿Avancé en el PROYECTO estructural?  (qué hice / qué falta)
   - ...
🎓 Materias: ¿al día? ¿algo en riesgo?
   - ...
💪 Físico: # entrenamientos esta semana: ___
🗣️ Inglés: # de clases/sesiones: ___
❤️ Relación: ¿tiempo de calidad? ¿algo que cuidar?
   - ...
🧠 Salud mental: ¿fui a psicología? ¿cómo me sentí?
   - ...
💼 Pasantía: aplicaciones enviadas: ___   respuestas: ___

🏆 Logro de la semana: _______________________________________
🎯 Foco #1 de la próxima semana: _____________________________
```

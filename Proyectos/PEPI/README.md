# 📘 PEPI — Proyecto

## Qué hay en esta carpeta
- [x] `PROYECTO PEPI.pdf` — Guía/enunciado de entregables
- [x] `INFORME_PEPI.md` — Informe completo desarrollado (todas las secciones)
- [x] `modelo_financiero.py` — Modelo financiero (WACC, TIR, VPN, flujos)
- [x] `resultados_modelo.csv` — Resultados y flujos de caja exportados

## Datos clave
- **Tema del proyecto:** Construcción de PTAP (120 L/s) y ampliación del sistema de acueducto — Municipio de El Progreso
- **Entregables exigidos:** Resumen ejecutivo, árbol de problemas, árbol de objetivos, MML, solución de ingeniería, análisis financiero (CAPEX/OPEX, deuda+equity, WACC, TIR, VPN, flujos), matriz de 20 riesgos, intervención de 10 riesgos (5 ambientales). Informe máx. 10 páginas.
- **Resultados financieros:** WACC 11,07 % · VPN proyecto COP 8.376 MM · TIR proyecto 18,13 % · TIR inversionista 24,78 %

## Cómo reproducir el modelo
```bash
python3 modelo_financiero.py   # imprime indicadores y genera resultados_modelo.csv
```

## Nota
El enunciado (PDF) es una **guía de otro proyecto**; este informe es un desarrollo de ejemplo
completo y consistente que sirve como plantilla para adaptar al proyecto real.

# Parcial - Diseño Estructural

Material de estudio para el parcial de Diseño Estructural (Concreto Reforzado, NSR-10).

## Archivos
- **`GUIA-PARCIAL.md`** → Guía completa con teoría explicada y todos los ejercicios resueltos
  paso a paso (Columnas, Provisiones sísmicas C.21, Escaleras, Zapatas y Vigas de amarre).

## Temas cubiertos
1. Columnas: cortas, flexo-compresión, diagrama de interacción, flexión biaxial, esbeltez.
2. Provisiones sísmicas para columnas (Capítulo C.21: DMO y DES).
3. Escaleras (Título K, tipologías, diseño).
4. Zapatas (cuadradas, rectangulares, excéntricas, combinadas).
5. Vigas de amarre (amarre sísmico, C.15.13).
